中文显示

之所以要搞中文，纯粹是因为自己的洋文道行不够！-_- (所以英语四六级还是要好好学的)
这个其实只要安装一个packages即可，这个包具体可看st3-zh_CN

1.从releases下载最新的Default.sublime-package
2.把它移动到/home/xx/.config/sublime-text-3/Installed Packages下(也可以打开st3->首选项->浏览程序包)
3.重启ST3

安装Package Control

Package Control是ST3的包管理工具，可以非常简单方便的搜索，安装以及更新包

自动安装

1.通过快捷键 ctrl+` 或者 View > Show Console 菜单打开控制台
2.粘贴对应版本的代码后回车安装

手动安装

1.Click the Preferences > Browse Packages… menu（）
2.Browse up a folder and then into the Installed Packages/ folder
3.Download Package Control.sublime-package and copy it into the Installed Packages/ directory
4.Restart Sublime Text

常用插件推荐

1.ctrl+shift+p调用出窗口：输入install package
2.然后输入需要的包名，回车即可安装。如:Markdown Preview
