联想ideapad-flex14黑苹果系统安装之路

发表于 2017-04-10 | 分类于 Hackintosh | 阅读次数

最近遇到了很多事情，真的是很多，可能是我能力不足，处理不过来吧，，还要加油才行。好了，希望被世界温柔以待。

Hackintosh，正式入门

flex14这台机子(配置图后面贴上)我安装了10.12.x后usb都识别在了3.0下，折腾了一下无果(暂时发现唯一的瑕疵就是QQ中看图放大后无法用鼠标移动看局部)，于是尝试装10.11.6，USB可以正常使用。如果有人知道如何解决这问题，欢迎再下方留言评论，谢谢。

我先安装个typora，写markdown方便点。

$ brew cask install typora

型号一样，配置可能不一样，以下表的为例：

部件 型号 说明
有线网卡 rtl8100
无线网卡 AR956x
核心显卡 Intel HD4400
独立显卡 nvidia GT820m
声卡 ALC283
蓝牙 AR3012
CPU intel core i5 4200u

其他的，比如硬盘(有固态的没搞过，但方法是有的)，内存，电源，usb什么的就不列了，用到的时候说明一下。

准备之系统镜像

镜像网上搜很多，质量参差不齐，可以自己做，我贴一个我用的10.11.6镜像,由ASIBA制作，在此表示感谢，百度云传送门,密码: a55f 。下载完毕后，把两个选中解压，得到镜像。

10.12.4的镜像由阿才提供，在此表示感谢，目的地在百度云盘传送门,密码：jmk9

准备之U盘制作

下载完毕后，记得核对MD5值，确认无误后，通过transmac把镜像刻到U盘里。

具体操作是在Windows中完成，这里提供一个transmac软件点我下载,密码：e5h6

（以下步骤图片随后附上)

软件安装完成后，以管理员身份运行。之后再软件的左侧栏会出现本地硬盘以及U盘（注意选对，好多人格错盘的）。

在U盘选项邮件弹出选择restore with disk image,然后选择你的mac镜像，点击刻录即可(时间有点久，可以喝杯茶什么的)

刻录完毕后。打开U盘，删掉clover>>driv64uefi目录下以csm开头的文件。然后配置clover.plist。这样U盘制作完毕。

准备之硬盘分区

需要准备两个地方，ESP分区和安装镜像的分区。

  1. esp分区：为硬盘引导分区，使用DiskGenius工具先备份下，以免出现问题无法恢复。要求分区>=240M,如果不够，需要手动扩大。鉴于联想的机子，一般出厂为260M，所以省了不少事。

  2. 安装镜像分区：从机子的任意分区压缩出一块分区来，不要太小(我分了100G)。先不要格式化磁盘，如果在后面的抹盘步骤找不到该分区的话，就来把它格成ntfs分区。

安装之bios设置

对于flex14，在BIOS中，需要调节以下几个地方：

deep s3 sleep： 开启。用来mac下的睡眠，不然会重启；

intel虚拟技术： 关闭。装了多系统，没必要了，当然开着也可以；

安全启动: 关闭。不然无法从U盘启动。

启动方式：uefi。当然开启传统支持也可以。

能想到的就这么多，，需要的时候补充

安装之系统安装

插入U盘，注意：插在usb2.0下，3.0的貌似有问题。选择从U盘启动，方式有三：

  1. 在windows下，设置>>系统更新及恢复>>系统恢复>>高级启动，然后选择从U盘启动。

  2. 关机状态下，按开机键后，按f12键，选择U盘启动。

  3. 关机状态下，按一键还原键，从boot menu中选择U盘启动

然后加载clover，出现系统启动选择项，首先选择第一项(有苹果标示)，随后加载完毕，出现安装界面，点back。

返回后出现语言选择界面，选择简体中文，下一步，

在左上角选择实用工具，磁盘工具，在左侧选中你事先准备好的分区，点击抹盘，名字自己起，格式选第一项日志式。如果找不到分区，回win下格成NTFS再来安装。

抹盘成功后，关掉磁盘工具，点击下一步，选中你的磁盘，继续安装，之后就是17分钟的等待，

安装之系统配置

顺利的经过17分钟后，自动重启，再次加载U盘clover，如果没有请手动启动。

这一次选择偏中间位置有一项boot os x from 磁盘名字，选中回车。继续加载。

然后需要注意的是，在网络配置界面选择不连接，如果你偏要选择，后果我不负责（哈哈）,其他的需要的时候补充。

当设定了用户名和密码后，不出意外，你所期望的mac os启动完毕。
